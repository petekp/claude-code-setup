## Principle
A user's first interaction with the product should demonstrate its value immediately by showing realistic sample content rather than presenting empty screens that require imagination.

## Why It Matters
New users cannot evaluate a product's usefulness when every screen is empty. An empty dashboard communicates nothing; a dashboard pre-populated with sample data communicates the product's structure, data model, and potential value within seconds. Sample content bridges the gap between "I signed up" and "I understand what this tool does for me" — the most critical and fragile moment in the user journey. Products that require users to do significant setup work before seeing any value suffer dramatically higher abandonment rates.

## Application Guidelines
- Pre-populate the product with realistic sample data (a sample project, example contacts, demo dashboard) that the user can explore, edit, and eventually delete
- Label sample content clearly (e.g., "[Sample] Q4 Marketing Campaign") so users can distinguish it from their own data and remove it when ready
- Design sample content to showcase the product's most valuable features — if the product excels at visualization, the sample data should produce an impressive chart, not a flat table
- Provide a clear, one-click path to dismiss or archive all sample content once the user is ready to work with their own data
- For B2B products, tailor sample content to the user's stated role or industry during signup when possible

## Anti-Patterns
- Dropping the user into a completely empty product with a "Get started" button and no indication of what the product will look like when populated
- Pre-populating with obviously fake data ("Test User," "Lorem Ipsum," "123 Main St.") that fails to communicate real-world value
- Making sample content indistinguishable from real data, causing confusion when users try to clean up their workspace
- Requiring the user to complete a lengthy setup wizard before seeing any product content, when sample data could have provided immediate orientation
