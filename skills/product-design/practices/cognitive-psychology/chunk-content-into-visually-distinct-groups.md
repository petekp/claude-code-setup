## Principle
Related content should be clustered into visually bounded groups using whitespace, borders, backgrounds, or cards so users can scan structure before reading detail.

## Why It Matters
Before reading any text, users perceive the visual structure of a page — they see blocks, gaps, and groupings. When content lacks visual chunking, every element appears to exist at the same level of importance, forcing users into slow linear reading. Visually distinct groups let users jump directly to the relevant section, dramatically reducing time-to-target. This is especially critical in information-dense applications like dashboards, settings panels, and admin tools where users repeatedly visit the same page for different purposes.

## Application Guidelines
- Use cards or bordered containers to group related data points (e.g., a user profile card containing name, email, and role together)
- Apply consistent spacing rules: tighter spacing within groups, wider spacing between groups to create clear visual separation
- Use subtle background color differences to distinguish sections on a single page (e.g., a light gray band for metadata, white for primary content)
- In forms, separate logical sections with horizontal rules, increased vertical spacing, or section headings — never rely solely on field order
- On dashboards, give each metric or widget its own bounded region so users can locate specific data by spatial memory
- Ensure the visual grouping hierarchy matches the informational hierarchy: primary groupings should be more visually prominent than subgroupings

## Anti-Patterns
- A dashboard where all metrics float in a single unstructured area with uniform spacing, making it impossible to scan for a specific category of information
- Form fields for "Billing Address" and "Shipping Address" in a continuous flow with only a label to separate them — no visual boundary
- A detail page where metadata, content, and actions all share the same visual treatment and spacing, creating an undifferentiated wall of information
